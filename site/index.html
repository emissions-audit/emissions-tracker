Why This Exists

~100 companies produce ~71% of global emissions. The data to hold them accountable exists but is scattered across PDFs, inconsistent formats, and paywalled databases costing $25K–$200K/year.

Cross-Validation

Compare what companies report against what satellites measure. Discrepancy flags surface greenwashing.

Multiple Sources

SEC EDGAR (regulatory), Climate TRACE (satellite), CDP (voluntary), and sustainability report PDFs — all normalized.

Open Source

MIT licensed. Self-host it, fork it, contribute to it. Raw data dumps available for free.

Developer-First

REST API with auto-generated Swagger docs. JSON + CSV export. API keys with generous free tier.

Try the API

List energy companies
curl https://api.emissions-tracker.org/v1/companies?sector=energy
Find major discrepancies between reported and measured emissions
curl https://api.emissions-tracker.org/v1/discrepancies?flag=red
Compare companies side-by-side
curl "https://api.emissions-tracker.org/v1/emissions/compare?companies=ID1,ID2&scopes=1,2&years=2022,2023"
Download full dataset
curl -H "X-API-Key: YOUR_KEY" https://api.emissions-tracker.org/v1/export/full?format=csv
