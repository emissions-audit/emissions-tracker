# Corrections Log

This file records all accepted community corrections to emissions figures in the database. Every correction is attributed to its contributor and linked to the primary source that justified the change.

Corrections are also exposed through the API: every emission row that has been corrected includes a `provenance` object listing contributors and field-level change history.

## How to submit a correction

Open a GitHub issue using the [Challenge a Number](.github/ISSUE_TEMPLATE/challenge-a-number.yml) template. See [CONTRIBUTING.md](CONTRIBUTING.md#challenge-a-number) for the full workflow.
